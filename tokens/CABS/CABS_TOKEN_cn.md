# CABS Token

## CABS 资讯
CABS Token 为固定利率之ABS

- 公开发售时间: 2017-07-14 01:00 UTC to 2017-08-04
- CABS年利率: 8%/每年
- CABS发行期: 3年期
- 配息频率: 每个月配息
- 配息日: 每个月5号
- 投资标的: 车贷
- 票面金额: USD (ETH/USD汇率为某个时间区间之平均值)
- 发行额度: 1,000,000 CABS (约等同1,000,000 USD)
- 交易闭锁期: 7天，至2017-08-11 UTC
- 计息起始: 2017-08-04 UTC

CABS发售期间，本公司保留随时中止购买之权利
