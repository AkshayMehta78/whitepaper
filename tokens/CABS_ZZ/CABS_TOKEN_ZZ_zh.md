# CABS Token

## CABS 資訊
CABS Token 為固定利率之ABS

- 公開發售時間: 2017-09-30 01:00 UTC to 2017-10-30
- CABS年利率: 6%/每年
- CABS發行期: 1年期
- 配息頻率: 期滿歸還本金與利息
- 配息日: 期滿配發
- 投資標的: 車貸
- 票面金額: USD (ETH/USD匯率為某個時間區間之平均值)
- 發行額度: 5,000,000 USD (相當於 180,000 CABS)
- 最低購買額度: 0.1 ETH
- 交易閉鎖期: 7天，至 2017-11-06 UTC
- 計息起始: 2017-11-06 UTC

CABS發售期間，本公司保留隨時中止購買之權利
